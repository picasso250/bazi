# “八字查询”Web 页面设计方案 (Stitch Design Pro) ## 1. 视觉方案与品牌调性 * **关键词**：专业、克制、东方、现代、学术。 * **设计理念**：将古老的历法天文数据通过现代排版技术呈现。借鉴宣纸、活字印刷的质感，结合现代 UI 的间距与响应式逻辑。 * **避雷点**：坚决不用发光特效、粗糙的八卦图、高饱和度红黄配色。 ## 2. 页面分区结构 ### 2.1 顶部 (Header) * **标题**：[八字真时查询] - 强调“真时”，体现学术严谨性。 * **描述**：基于地理经纬度精准换算平太阳时与真太阳时，推算四柱干支。 ### 2.2 输入区 (Input Section) - 模块化表单 * **出生时间模块**：年、月、日、时、分（下拉或数字输入）。 * **出生地点模块**： * 国内：省/市/区 联动选择。 * 海外：城市名称输入。 * *提示文案*：“地点用于计算地理经度，以获取精准的真太阳时。” * **性别选择**：简洁的 Segmented Control（分段开关）。 * **操作**：大尺寸、稳重的“开启查询”按钮。 ### 2.3 结果展示区 (Result Section) * **核心摘要卡片**：展示换算后的当地时间、经纬度、太阳黄经。 * **四柱核心模块**：四张并列卡片（年、月、日、时），干支字体加粗放大，十神作为角标或底部辅助。 * **藏干详情**：紧跟在四柱卡片下方，采用整齐的网格布局。 * **大运流年**：采用横向滑动的“时间轴”或“卡片流”，区分起运岁数与当前状态。 ## 3. 设计规范建议 * **色彩 (Palette)**： * 背景：`#F9F8F4` (暖米白) * 文字：`#2C2C2C` (墨黑) * 主色：`#3A5B6F` (黛青) * 装饰：`#B89F77` (铜金) * 警示：`#A63F3F` (深红) * **字体 (Typography)**： * 标题/干支：优先使用衬线体（如思源宋体），体现文化底蕴。 * 正文/数据：非衬线体（如 PingFang SC），确保屏显清晰度。 * **间距 (Spacing)**：遵循 8px 栅格系统，保持充分的间距（Whitespace）。 ## 4. UI 文案建议 (更专业化) * “查询” -> **“开启推演”** 或 **“生成报告”** * “输入出生信息” -> **“录入生辰信息”** * “结果” -> **“命理参数概要”** * 错误提示：“请选择城市” -> **“需确定出生位置以计算真太阳时”** ## 5. 交互建议 * **Loading**：点击查询后，展示一个简约的“天文轨道旋转”加载动画，持续 1s 左右，增加仪式感。 * **Reveal**：结果区由下至上平滑淡入，视线聚焦在四柱卡片上。 The Celestial Scholar | 八字真时查询

何谓真太阳时？

在传统八字计算中，时间的精确度至关重要。平太阳时（如北京时间）是基于行政划分的人造时间，而真太阳时则是基于您出生地点相对于太阳的真实位置。本系统通过高精度经纬度算法，为您还原最准确的天文时空坐标。
